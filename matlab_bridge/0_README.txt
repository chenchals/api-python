0_README.txt for NWB matlab_bridge API

The matlab_bridge implements a matlab API to the NWB format by interfacing
matlab to the Python NWB API.

See file 0_INSTALL.txt for instructions about how to install the matlab_bridge
API.

Directories are:

matlab_examples  - matlab scripts used to create sample NWB files

matlab_unittest  - matlab unittests

matlab_bridge_api - software implementing the matlab bridge API
