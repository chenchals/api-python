This directory is for contining source data for the NWB matlab example
script "crcns_alm_1.m".
To run the script, the contents of this directory (e.g.
directory "crcns_alm-1" and the .mat and .txt files inside)
must be placed in the matlab_bridge/matlab_examples/source_data
directory.  (Or this directory can be symbolically linked to
that directory).
See file 0_README.txt in the directory above this for
instructions for getting the source data files.
